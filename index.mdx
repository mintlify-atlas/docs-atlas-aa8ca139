<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with GUN in minutes.
  </p>

  <Steps>
    <Step title="Install GUN">
      Install GUN via npm, yarn, or use the CDN link directly in your HTML.
      
      <CodeGroup>
      ```bash npm
      npm install gun
      ```
      ```bash yarn
      yarn add gun
      ```
      ```bash pnpm
      pnpm add gun
      ```
      </CodeGroup>
      
      Or include via CDN:
      ```html
      <script src="https://cdn.jsdelivr.net/npm/gun/gun.js"></script>
      ```
    </Step>

    <Step title="Initialize GUN">
      Create a GUN instance to start syncing data across peers.
      
      ```javascript
      // In Node.js
      const Gun = require('gun');
      const gun = Gun();
      
      // In browser (after CDN include)
      const gun = Gun();
      ```
      
      <Tip>GUN works offline-first. Data syncs automatically when peers connect.</Tip>
    </Step>

    <Step title="Store and retrieve data">
      Use GUN's graph API to put and get data with realtime updates.
      
      ```javascript
      // Store data
      gun.get('users').get('alice').put({
        name: 'Alice',
        age: 25
      });
      
      // Subscribe to realtime updates
      gun.get('users').get('alice').on((data, key) => {
        console.log('Alice:', data);
      });
      ```
      
      <Note>Changes propagate in realtime to all connected peers automatically.</Note>
    </Step>

    <Step title="Explore advanced features">
      Add authentication, encryption, and P2P networking to your app.
      
      ```javascript
      // User authentication with SEA
      const user = gun.user();
      user.create('alice', 'password', (ack) => {
        if(ack.err) return console.log(ack.err);
        user.auth('alice', 'password');
      });
      ```
    </Step>
  </Steps>
</div>
